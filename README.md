### ✨ Caratteristiche Principali
* **Accettazione & Ritiro**: Registra i dispositivi in ingresso con checklist di test (tasti, touch, fotocamere), dati cliente, pin o segno di sblocco e priorità.
* **Preventivi Personalizzati**: Genera al volo preventivi di riparazione commerciali o configurazioni PC personalizzate (Gaming, Ufficio, ecc.).
* **Modificatore PDF Visivo**: Editor grafico a coordinate con zoom regolabile (50% - 200%) e controlli colore completi per le tabelle.
* **Gestione Cassa & Ricevute**: Gestione rapida degli acquisti con storico delle transazioni.
* **Report Collaudo**: Stampe dedicate per i test post-riparazione con esiti colorati.
